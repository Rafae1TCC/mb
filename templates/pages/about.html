About this message board

This message board is a simple Django application designed to allow users to post and view messages. It serves as a demonstration of basic CRUD (Create, Read, Update, Delete) operations in Django.

Features

  • Post new messages
  • View all messages
  • Edit existing messages
  • Delete messages

Technologies Used

  • Django
  • Bootstrap for styling
  • SQLite for the database
